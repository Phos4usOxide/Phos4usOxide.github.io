Animations

Animation is the illusion of movement by rapidly displaying static images (frames) in a sequence. Animations are often viewed on interfaces as videos, which are discussed on the "Lab 5 - Videos" blog page found here. These animations have specific framerates, like videos, so the more frames per second, the faster it cycles through them, ultimately making the animation smoother. Animation examples are shown below in videos. Let's check out some information and guidelines for animation usage!

Video by VideoPlasty.com from Pixabay

Creation

Depending on the type of animation, the frames are created in different ways, such as drawing or rendering. When an animation is created using rendering, there are usually key frames that store object states at significant points during the sequence, and then a process called "tweening" will calculate the frames between these keypoints to match the frames per second. Another method instead of key frames and tweening are scripts that run transformation commands on the objects. When using animations, make sure the frame rate is high enough for the animation to look smooth. A good amount would be at least 24 which is the standard for movies.

Types

Animations can be made in both 2D and 3D space and can be hand-drawn or computer-generated. Some types of animations include:

  • Traditional: drawing each frame by hand and putting them in a sequence
  • 2D Animation: vector-based made using computers
  • 3D Animation: made in 3D space with advanced modelling software
  • Motion Graphics: graphical elements are moved and animated
  • Stop Motion: moving and taking pictures of objects and putting them in a sequence
  • Rotoscoping: capturing human movements and drawing over it

Usage

Animations are used in many multimedia scenarios for various reasons. They can allow information to be easily understood and visualized (better than text) or used whenever simulation of movement is needed. Some examples include:

  • educational, scientific, or medical videos to help visualize ideas
  • showcases or advertisements for products and services
  • entertainment such as movies or videos
  • transitions between presentation slides
  • movement for characters and other dynamic objects in video games
